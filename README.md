# CINEMATA — PHP + JSON ფილმების საიტი (MySQL-ის გარეშე)

სრული ვებ-გვერდი ფილმებისთვის/სერიალებისთვის: მთავარი გვერდი ჰეროში ჩაშენებული
ტრენდულებით, კვადრატული ბარათები, ცალკე ფილტრის გვერდი ჩექბოქსებით, სლაიდერებით,
ჭკვიანი ძიებითა და infinite scroll-ით, TikTok-სტილის „Scroll Movie“, და სრული
ადმინ პანელი. მონაცემთა ბაზა — JSON ფაილები (MySQL არ სჭირდება).

## 🚀 გაშვება ჰოსტინგზე

1. ატვირთე **ყველა ფაილი** ჰოსტინგის root საქაღალდეში (მაგ. `public_html/`).
2. გახსენი საიტი ბრაუზერში — `index.php` ავტომატურად ჩატვირთავს მთელ საიტს.
   პირველ გახსნაზე `data/` საქაღალდეში თავისით შეიქმნება JSON ბაზა დემო-მონაცემებით.
3. დარწმუნდი, რომ `data/` და `uploads/` საქაღალდეებში ჩაწერა შესაძლებელია
   (უფლებები `0775` ან `0755`). cPanel-ში: File Manager → Permissions.

PHP 7.4+ (რეკომენდებული 8.x). სხვა გაფართოება/ბაზა საჭირო არ არის.

## 🔑 ადმინ პანელი

- მისამართი: `/admin/` (ან `/admin/login.php`)
- ნაგულისხმევი: **admin / admin123**
- შესვლისთანავე შეცვალე პაროლი: ადმინ პანელი → **პარამეტრები**.

ადმინიდან აკონტროლებ ფრონტ-ის ყველაფერს:
- **ფილმები** — დამატება/რედაქტირება/წაშლა. ყველა ველი: ტიპი, ჟანრები, გახმოვანება,
  ქვეყანა, კოლექცია, წელი, რეიტინგი, ხანგრძლივობა, რეჟისორი, მსახიობები, აღწერა.
- **სურათები/ვიდეო** — ბმულით (URL) **ან** პირდაპირი ატვირთვით (mp4/webm).
- **განთავსება** — ჩექბოქსით აირჩევ: ტრენდული (ჰეროს რეილში), ჰეროს სლაიდი (featured),
  ბოლოს ნანახი (პროგრესით).
- **კატეგორიები** — ფილტრის ყველა მენიუს მართვა (ტიპები, ჟანრები, გახმოვანება,
  ქვეყნები, კოლექციები).
- **Scroll Movie** — ცალკე სივრცე reels-ვიდეოების დასამატებლად (URL ან mp4 ატვირთვა).
- **პარამეტრები** — საიტის სათაური, ადმინის მომხმარებელი/პაროლი.

## 🗂️ სტრუქტურა

```
index.php            მთავარი — კომპონენტებად აწყობილი
filter.php           ცალკე ფილტრის გვერდი (იხსნება მხოლოდ ხელით)
movie.php            ფილმის დეტალური გვერდი + დაკვრა
scroll-movie.php     TikTok-სტილის reels
config.php           კონფიგი, ბილიკები, ჰელფერები
lib/db.php           JSON ბაზა + ფილტრის ძრავა
lib/seed.php         პირველი გაშვების დემო-მონაცემები
lib/upload.php       სურათის/ვიდეოს ატვირთვა
includes/            ფრონტ-კომპონენტები (header, hero, collections, recent, footer, render)
api/movies.php       JSON ენდპოინტი (ფილტრი + ძიება + გვერდები)
assets/css|js        სტილები და სკრიპტები
admin/               ადმინ პანელი
data/                JSON ბაზა (ავტომატურად იქმნება) — დაცულია .htaccess-ით
uploads/             ატვირთული სურათები/ვიდეო — სკრიპტი არ სრულდება (.htaccess)
```

## ✨ მთავარი ფუნქციები

- **ჰერო + ტრენდულები ერთად** — ტრენდულები ჰეროს შიგნითაა, ქვედა ფენაში, აღწერას
  ხელს არ უშლის არცერთ მოწყობილობაზე.
- **კვადრატული ბარათები** — პოსტერები 1:1 თანაფარდობით.
- **ფილტრი ცალკე გვერდად** — სანამ ხელით არ გახსნი, არ იტვირთება; გახსნისას მთელ
  ეკრანს იკავებს (დანარჩენი საიტი „იხურება“); × ღილაკი აბრუნებს მთავარზე.
- **ჭკვიანი ძიება** — ეძებს ნებისმიერ ველში (სათაური, ჟანრი, ქვეყანა, აღწერა…).
- **Infinite scroll + lazy loading** — სურათები `loading="lazy"`-ით, ბარათები
  პორციებად, რომ საიტი არ დაიმძიმოს.
- **დუალ-სლაიდერები** — რეიტინგი (0–10) და წელი.

## ⚙️ შენიშვნები

- დიდი ვიდეოს ატვირთვისთვის შესაძლოა საჭირო გახდეს ჰოსტინგზე `upload_max_filesize`
  და `post_max_size` გაზრდა (root `.htaccess`-ში უკვე მითითებულია; ზოგ ჰოსტზე
  php.ini-დან უნდა შეიცვალოს).
- JSON ბაზის სარეზერვო ასლი = `data/` საქაღალდის კოპირება.
